## What is Slide?

Slide is a comprehensive AI agent framework that provides everything you need to build intelligent, autonomous applications. It consists of four powerful packages that work seamlessly together:

<CardGroup cols={2}>
  <Card
    title="Tyler"
    icon="robot"
    href="/packages/tyler/introduction"
  >
    Core agent framework with MCP (Model Context Protocol) support for building autonomous AI agents
  </Card>
  <Card
    title="Lye"
    icon="toolbox"
    href="/packages/lye/introduction"
  >
    Rich utility library providing tools for audio, browser automation, image processing, and more
  </Card>
  <Card
    title="Narrator"
    icon="database"
    href="/packages/narrator/introduction"
  >
    Thread and storage management system for maintaining conversation history and agent state
  </Card>
  <Card
    title="Space Monkey"
    icon="slack"

  >
    Slack integration framework for deploying AI agents as Slack bots
  </Card>
</CardGroup>

## Quick Example

```python
from tyler import Agent
from lye import audio, browser

# Create an agent with audio and browser capabilities
agent = Agent(
    name="research-assistant",
    tools=[audio.transcribe, browser.screenshot, browser.extract_text]
)

# Run the agent
result = await agent.run(
    "Transcribe this podcast and create a summary with screenshots"
)
```
