Space Monkey makes it easy to deploy AI agents as Slack bots. It handles all the complexity of Slack's APIs, event handling, and message formatting, so you can focus on building great agent experiences.

## Overview

Space Monkey provides:
- **Easy Setup**: Get a Slack bot running in minutes
- **Event Handling**: Respond to messages, mentions, and commands
- **Rich Messaging**: Support for blocks, attachments, and interactive elements
- **Thread Management**: Automatic conversation threading
- **Async Support**: Built for high-performance async operations

## Installation

```bash
uv add slide-space-monkey
```

## Quick Start

```python
from space_monkey import SlackApp
from tyler import Agent

# Create your agent
agent = Agent(
    name="slack-assistant",
    model_name="gpt-4",
    purpose="To help the team with questions and tasks"
)

# Create Slack app
app = SlackApp(
    agent=agent,
    slack_bot_token="xoxb-your-bot-token",
    slack_app_token="xapp-your-app-token"
)

# Run the bot
if __name__ == "__main__":
    app.start()
```

## Slack App Setup

### 1. Create a Slack App

1. Go to [api.slack.com/apps](https://api.slack.com/apps)
2. Click "Create New App" â†’ "From an app manifest"
3. Use this manifest:

```yaml
display_information:
  name: My AI Assistant
features:
  bot_user:
    display_name: AI Assistant
oauth_config:
  scopes:
    bot:
      - channels:history
      - channels:read
      - chat:write
      - groups:history
      - groups:read
      - im:history
      - im:read
      - mpim:history
      - mpim:read
settings:
  event_subscriptions:
    bot_events:
      - app_mention
      - message.channels
      - message.groups
      - message.im
      - message.mpim
  interactivity:
    is_enabled: true
  org_deploy_enabled: false
  socket_mode_enabled: true
```

### 2. Install to Workspace

1. Click "Install to Workspace"
2. Copy the Bot User OAuth Token
3. Go to "Basic Information" â†’ "App-Level Tokens"
4. Create a token with `connections:write` scope

### 3. Configure Your Bot

```python
app = SlackApp(
    agent=agent,
    slack_bot_token="xoxb-...",  # Bot User OAuth Token
    slack_app_token="xapp-...",   # App-Level Token
)
```

### App Mentions

```python
@app.event("app_mention")

    # Agent responds when mentioned
    pass
```

### Commands

```python

    # Process command
```
