## Common Patterns

### Async-First Design

All Slide APIs are designed to be async-first:

```python
# All main operations are async
result = await agent.run("Task")
data = await tool.execute(params)
thread = await Thread.create()
```

### Consistent Error Handling

All packages follow a consistent error hierarchy:

```python
from tyler.exceptions import TylerError, AgentError, ToolError
from lye.exceptions import LyeError, ToolExecutionError
from narrator.exceptions import NarratorError, StorageError
from space_monkey.exceptions import SpaceMonkeyError, SlackError
```

### Type Safety

All APIs are fully typed with Python type hints:

```python
from typing import List, Dict, Optional
from tyler import Agent
from lye.tools import Tool

async def create_agent(
    name: str,
    tools: List[Tool],
    model: Optional[str] = None
) -> Agent:
    ...
```

## API Conventions

### Naming Conventions

- **Classes**: PascalCase (e.g., `Agent`, `MCPAdapter`, `FileStore`)
- **Functions**: snake_case (e.g., `create_agent`, `run_tool`)
- **Constants**: UPPER_SNAKE_CASE (e.g., `DEFAULT_MODEL`, `MAX_RETRIES`)
- **Private**: Leading underscore (e.g., `_internal_method`)

### Method Types

1. **Instance Methods**: Called on object instances
   ```python
   agent = Agent(name="bot")
   result = await agent.run("task")
   ```

2. **Class Methods**: Called on the class itself
   ```python
   thread = await Thread.from_id("thread-123")
   ```

3. **Static Methods**: Utility functions
   ```python
   is_valid = Tool.validate_parameters(params)
   ```

### Return Values

Most APIs follow consistent return patterns:

```python
# Single values
result: str = await agent.run("task")

# Structured data
response: Dict[str, Any] = await tool.execute(params)

# Lists
tools: List[Tool] = await adapter.list_tools()

# Optional values
thread: Optional[Thread] = await Thread.get_by_id(id)
```

## Authentication & Configuration

### API Keys

Set API keys via environment variables:

```bash
export OPENAI_API_KEY="sk-..."
export ANTHROPIC_API_KEY="sk-ant-..."
export SLACK_BOT_TOKEN="xoxb-..."
```

Or pass directly:

```python
agent = Agent(
    name="bot",
    api_key="sk-...",  # Not recommended
    model="gpt-4"
)
```

### Configuration Files

Tyler supports configuration files:

```yaml
# tyler-config.yaml
agents:
  default:
    model: gpt-4
    temperature: 0.7
    
tools:
  enabled:
    - web.search
    - files.read
    
mcp:
  servers:
    - stdio://./tools-server
```

## Rate Limiting

Most LLM providers have rate limits. Slide handles these automatically:

```python
# Automatic retry with exponential backoff
agent = Agent(
    name="bot",
    retry_config={
        "max_retries": 3,
        "initial_delay": 1.0,
        "max_delay": 60.0,
        "exponential_base": 2
    }
)
```

## Debugging

Enable debug logging:

```python
import logging
from tyler.utils.logging import setup_logging

# Enable debug logs
setup_logging(level=logging.DEBUG)

# Or use environment variable
# SLIDE_LOG_LEVEL=DEBUG
```

## API Versioning

Slide follows semantic versioning:

- **Major**: Breaking changes (1.0.0 → 2.0.0)
- **Minor**: New features, backwards compatible (1.0.0 → 1.1.0)
- **Patch**: Bug fixes (1.0.0 → 1.0.1)

Check version:

```python
import tyler
print(tyler.__version__)  # "1.0.0"
```

## Quick Reference

### Tyler

```python
from tyler import Agent
from tyler.mcp import MCPAdapter

# Create agent
agent = Agent(name="bot", tools=[...])
result = await agent.run("task")

# MCP integration
adapter = MCPAdapter()
await adapter.connect("stdio://server")
```

### Lye

```python
from lye import audio, browser, files, image, web

# Use tools
transcript = await audio.transcribe("file.mp3")
screenshot = await browser.screenshot("https://example.com")
content = await files.read("data.txt")
```

### Narrator

```python
from narrator import Thread, FileStore

# Create thread
store = FileStore("./data")
thread = Thread(store=store)
await thread.add_message("user", "Hello")
```

### Space Monkey

```python
from space_monkey import SlackApp

# Create Slack app
app = SlackApp(agent=agent)
app.start(port=3000)
```
