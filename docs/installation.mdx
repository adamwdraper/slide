## Quick Install

The fastest way to get started with Slide is to install all core packages:

<Tabs>
  <Tab title="uv (Recommended)">
    ```bash
    uv add slide-tyler slide-lye slide-narrator
    ```
  </Tab>
  <Tab title="pip">
    ```bash
    pip install slide-tyler slide-lye slide-narrator
    ```
  </Tab>
</Tabs>

## Prerequisites

### Python Version
Slide requires Python 3.11 or higher.

### API Keys
You'll need an API key for your chosen LLM provider:

<AccordionGroup>
  <Accordion title="OpenAI">
    ```bash
    export OPENAI_API_KEY="sk-..."
    ```
    [Get your API key →](https://platform.openai.com/api-keys)
  </Accordion>
  
  <Accordion title="Anthropic">
    ```bash
    export ANTHROPIC_API_KEY="sk-ant-..."
    ```
    [Get your API key →](https://console.anthropic.com/keys)
  </Accordion>
  
  <Accordion title="Google">
    ```bash
    export GOOGLE_API_KEY="..."
    ```
    [Get your API key →](https://makersuite.google.com/app/apikey)
  </Accordion>
</AccordionGroup>

## Package Options

### Core Agent Package
```bash
# Just the agent framework
uv add slide-tyler
```

### With Tools
```bash
# Agent + tool library
uv add slide-tyler slide-lye
```

### With Memory
```bash
# Agent + persistent storage
uv add slide-tyler slide-narrator
```

### Full Installation
```bash
# Everything including Slack integration
uv add slide-tyler slide-lye slide-narrator slide-space-monkey
```

## Verify Installation

Create a test file to verify everything is working:

```python
import asyncio
from tyler import Agent, Thread, Message

async def main():
    agent = Agent(
        name="test-agent",
        model_name="gpt-3.5-turbo",
        purpose="To test the installation"
    )
    
    thread = Thread()
    message = Message(role="user", content="Say hello!")
    thread.add_message(message)
    
    processed_thread, new_messages = await agent.go(thread)
    
    for msg in new_messages:
        if msg.role == "assistant":
            print(f"✅ Installation successful! Agent says: {msg.content}")

if __name__ == "__main__":
    asyncio.run(main())
```

Run the test:
```bash
python test_installation.py
```

## Development Setup

### Using UV (Recommended)

UV is a fast Python package manager that we recommend for Slide projects:

```bash
# Install uv
curl -LsSf https://astral.sh/uv/install.sh | sh

# Create a new project
uv init my-agent-project
cd my-agent-project

# Add Slide packages
uv add slide-tyler slide-lye slide-narrator

# Add development dependencies
uv add --dev pytest pytest-asyncio ruff
```

### Using Virtual Environments

```bash
# Create virtual environment
python -m venv venv

# Activate it
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install packages
pip install slide-tyler slide-lye slide-narrator

# Install dev dependencies
pip install pytest pytest-asyncio ruff
```

## Optional Dependencies

### Database Support

For PostgreSQL support with Narrator:
```bash
uv add asyncpg
```

For SQLite support (included by default):
```bash
uv add aiosqlite
```

### Additional Tools

Some Lye tools require extra dependencies:

```bash
# For browser automation
uv add playwright
playwright install chromium

# For audio processing
uv add pydub

# For advanced image processing
uv add pillow
```

## Environment Configuration

Create a `.env` file in your project root:

```bash
# LLM Provider Keys
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...

# Optional: Slack Bot (for Space Monkey)
SLACK_BOT_TOKEN=xoxb-...
SLACK_SIGNING_SECRET=...

# Optional: Database URL (for Narrator)
DATABASE_URL=postgresql://user:pass@localhost/dbname

# Optional: File Storage Path
FILE_STORAGE_PATH=./agent_files
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Module Not Found Error">
    If you get import errors, ensure you've activated your virtual environment and installed the packages:
    ```bash
    # Check installed packages
    pip list | grep slide
    
    # Reinstall if needed
    pip install --upgrade slide-tyler slide-lye slide-narrator
    ```
  </Accordion>
  
  <Accordion title="API Key Errors">
    Ensure your API keys are properly set:
    ```bash
    # Check if key is set
    echo $OPENAI_API_KEY
    
    # Set it if missing
    export OPENAI_API_KEY="your-key-here"
    ```
  </Accordion>
  
  <Accordion title="Async/Await Errors">
    Slide uses async/await. Make sure you're using Python 3.11+ and running code with asyncio:
    ```python
    import asyncio
    
    async def main():
        # Your code here
        pass
    
    asyncio.run(main())
    ```
  </Accordion>
</AccordionGroup>
